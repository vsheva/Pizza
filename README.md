# Pizza Online Store 
<hr>

This project is [here](https://profound-otter-73a6e3.netlify.app/).

## The tech stack is:
- ReactJS 18
- TypeScript
- Redux Toolkit 
- React Router v6 
- Axios + Fetch 
- React Hooks 
- Prettier 
- CSS-Modules / SCSS 
- React Content Loader (sceleton)
- React Pagination 
- Lodash.Debounce
- Code Splitting, React Loadable, useWhyDidYouUpdate

<hr>

In the project directory, you can run:
#### `npm start`
